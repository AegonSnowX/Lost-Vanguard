##Project Lynora (Working Title)

Project Lynora is a 2D top-down sci-fi game focused on exploration, interaction, and consequence-driven progression.

The game places the player in a hostile, hidden star system where survival, discovery, and moral tension intersect. It is not about conquest or heroism, but about operating under pressure in a world that actively resists being found.
